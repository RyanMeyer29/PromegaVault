#My-Projects

This is a potential collaboration with Jack Miller to test and help develop an AI tool that generates SolidWorks drawings automatically from 3D models. As of the biweekly product finishing huddle on 2026-03-23, the tool is consistently applying the correct drawing template, selecting the right views, and populating title block information. The main outstanding issue is dimensioning — it is not reliably placing dimensions, particularly on smaller parts, and sometimes skips them entirely rather than prompting for manual input. Jack and Ali Heinemann have both been working on this.

The suggestion from the meeting was that I try the tool on my own projects to get more reps on it, and potentially collaborate with Jack directly — possibly sharing the same baseline so we can both work on solving the dimensioning problem together.
