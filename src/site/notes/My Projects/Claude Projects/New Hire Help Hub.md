#My-Projects

The New Hire Help Hub is a concept for a centralized SharePoint page that brings together different co-op-built tools and resources to help new hires get oriented at Promega. It is mostly still an idea at this stage — the vision is a combination of different co-op projects rather than a single tool built by one person. The Gizmos and Gadgets Glossary is expected to be one of the core resources linked from this hub.

## Co-op Contributions (Planned)

- Ryan — Gizmos and Gadgets Glossary
- Will — Co-op planner
- Ethan — Skill tester / SharePoint integration
- Adiva — SAP Glossary
- Amanda — CAD advisor (possibly a specialized chatbot for machinability and design steps)

## Future Goals

- Build out the SharePoint page structure
- Link co-op tools and projects as they are completed
- Eventually serve as the go-to landing page for incoming co-ops
