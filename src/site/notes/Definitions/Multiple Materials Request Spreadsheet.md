#Definitions

The Multiple Materials Request Spreadsheet is an Excel-based form used to request SAP part numbers for 5 or more materials at once. It lives on SharePoint in the Plant Maintenance/FEEP area and is maintained by Nikki Masterson. For fewer than 5 parts, the Smartsheet-based Parts Order Form is used instead.

The spreadsheet has five tabs, each corresponding to a different request type: Existing SAP ERSA Material, Existing SAP NLAG Material, New SAP ERSA Material, New SAP NLAG Material, and No SAP Material. The New SAP ERSA Material tab is the most relevant for adding newly machined or purchased parts to the system. Its columns are: SAP Material Description, Automated Reordering, Reorder Point (if Automated Reordering = Yes), Rounding Value (if Automated Reordering = Yes), Equipment BOM Information, Vendor Material Number, Order Quantity, Unit of Measure, Price per Unit, Storage Location, Quantity on Hand, and Storage Bin. Fields marked as required must be completed to submit the request; the remaining columns are optional SAP data updates.
