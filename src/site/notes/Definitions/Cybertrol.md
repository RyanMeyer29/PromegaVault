#Definitions

Cybertrol is an external contractor working with Promega to configure and set up the Batch control system on the fermenters in the Arnold Center. A small team from Cybertrol has access to the [[Software/Smartsheet\|Smartsheet]] bug tracking list used during Batch Run Testing to log issues found during recipe verification.
